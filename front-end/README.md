# SCAMPI

## Starting the front-end in development on http://127.0.0.1:5000/
From Visual-Bioinformatics/front-end/ run:
* `$ ng serve --open`

## Unit testing the front-end
From Visual-Bioinformatics/front-end/ run:
* `$ ng test`

## Linting the back-end
From Visual-Bioinformatics/front-end/ run:
* `$ ng lint`
